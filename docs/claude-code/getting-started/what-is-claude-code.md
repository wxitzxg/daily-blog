# 什么是 Claude Code

> Claude Code 是 Anthropic 推出的 AI 编程助手，专注于提升开发者效率。

## 核心理念

Claude Code 不只是一个代码补全工具，而是一个**完整的 AI 编程工作流系统**。它通过以下方式改变开发：

- **智能对话**：理解项目上下文，进行有意义的对话
- **技能系统（Skills）**：封装可复用的专业技能
- **Agent 模式**：自主执行复杂任务
- **MCP 服务**：连接外部工具和服务

## 解决什么问题

| 痛点 | Claude Code 解决方案 |
|------|---------------------|
| 重复代码手动编写 | Agent 自动生成 |
| 配置每次重写 | Skills 复用 |
| 工具割裂 | MCP 一站式连接 |
| 上下文丢失 | Memory 持久化 |

## 快速开始

```bash
# 安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 初始化项目
claude init

# 开始对话
claude chat
```
