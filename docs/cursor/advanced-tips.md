# Cursor 进阶技巧

> 提升使用效率

## 进阶功能

### 1. Tab 补全

连续按 Tab 接受建议：
```
Tab → 接受第一个建议
Tab Tab → 接受多个建议
```

### 2. Ctrl+K 命令

快捷生成代码：
```
Ctrl+K → 生成代码
```

### 3. Ctrl+L 对话

唤起 AI 对话：
```
Ctrl+L → 打开对话窗口
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+K | 生成代码 |
| Ctrl+L | AI 对话 |
| Ctrl+/ | 解释代码 |
| Ctrl+Shift+R | 重构 |
