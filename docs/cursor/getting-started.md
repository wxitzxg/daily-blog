# Cursor 基础入门

> AI 增强的代码编辑器

## 核心理念

Cursor 是基于 VS Code 的 AI 代码编辑器，集成了 GPT 模型，提供智能代码补全、生成和解释功能。

## 主要特点

| 功能 | 说明 |
|------|------|
| Code Completion | 智能代码补全 |
| Chat | 对话式代码生成 |
| Explain | 代码解释 |
| Refactor | 代码重构 |

## 快速开始

1. 下载安装：cursor.sh
2. 登录账号
3. 开始编码

## 与 Claude Code 对比

| 方面 | Cursor | Claude Code |
|------|--------|-------------|
| 界面 | GUI | 终端 |
| 交互 | 即时补全 | 对话式 |
| 适用 | 小任务 | 复杂项目 |
