# Superpowers 案例实战

> 真实场景应用

## 案例 1：产品功能开发

```
任务：开发用户画像功能

第一步：Questioning
- 需要哪些数据？
- 用户画像用途？
- 数据来源？

第二步：Planning
- 数据采集模块
- 画像计算服务
- API 接口
- 前端展示

第三步：Coding
- 实现各个模块

第四步：Testing
- 单元测试
- 集成测试

第五步：Code Review
- 代码质量检查
- 安全性审查
```

## 案例 2：Bug 修复

```
任务：修复登录超时问题

第一步：Debugging
- 收集错误日志
- 分析调用链路
- 定位问题

第二步：Coding
- 编写修复代码

第三步：Testing
- 验证修复

第四步：Documenting
- 记录问题和解决方案
```

## 案例 3：技术调研

```
任务：调研新技术方案
```
