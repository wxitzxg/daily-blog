# Superpowers 核心技能详解

> 深入理解关键技能

## 1. Brainstorming 技能

**用途：** 头脑风暴，激发创意

```
使用 Brainstorming：
- 帮我构思一个新功能
- 设计一个系统架构
- 解决一个技术难题
```

## 2. Planning 技能

**用途：** 制定实施计划

```
使用 Planning：
- 帮我制定一个项目计划
- 分解这个大任务
- 规划开发路线
```

## 3. Code Review 技能

**用途：** 深度代码审查

```
使用 Code Review：
- 审查这个模块
- 检查安全问题
- 给出改进建议
```

## 4. Debugging 技能

**用途：** 系统性调试

```
使用 Debugging：
- 帮我分析这个 bug
- 找到问题根因
- 制定修复方案
```

## 技能组合

### 场景：开发新功能

```
1. Brainstorming → 构思功能
2. Planning → 制定计划
3. Coding → 实现代码
4. Testing → 生成测试
5. Code Review → 审查代码
```
