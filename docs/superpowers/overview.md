# Superpowers 技能系统概述

> Claude Code 的技能增强系统

## 核心理念

Superpowers 是 Claude Code 的技能系统扩展，提供一系列预定义的 Skills 来增强 AI 编程能力。

## 14 个核心技能

| 技能 | 功能 |
|------|------|
| Brainstorming | 头脑风暴 |
| Coding | 代码编写 |
| Code Review | 代码审查 |
| Debugging | 调试排错 |
| Designing | 系统设计 |
| Documenting | 文档编写 |
| Explaining | 代码解释 |
| Planning | 计划制定 |
| Questioning | 需求澄清 |
| Reading | 代码阅读 |
| Researching | 技术调研 |
| Reviewing | 审查评审 |
| Testing | 测试生成 |
| Writing | 写作输出 |

## 使用方法

```
使用 Brainstorming 技能帮我想一个新功能
使用 Code Review 技能审查这段代码
```
