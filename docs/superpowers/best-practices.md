# Superpowers 最佳实践

> 高效使用技能

## 使用技巧

### 1. 选择正确的技能

| 任务类型 | 推荐技能 |
|----------|----------|
| 创意激发 | Brainstorming |
| 需求分析 | Questioning |
| 实现代码 | Coding |
| 问题排查 | Debugging |
| 质量保证 | Code Review |

### 2. 组合使用

```
复杂任务流程：
Questioning → 澄清需求
  ↓
Planning → 制定计划
  ↓
Coding → 实现
  ↓
Testing → 测试
  ↓
Code Review → 审查
```

### 3. 迭代优化

```
每轮迭代：
1. 使用技能完成任务
2. 评估结果
3. 选择下一个技能
4. 继续优化
```

## 常见场景

### 场景 1：开发新功能

```
1. Questioning: 了解需求细节
2. Planning: 制定实现计划
3. Coding: 编写代码
4. Testing: 生成测试
```

### 场景 2：重构代码

```
1. Reading: 理解现有代码
2. Planning: 制定重构计划
3. Coding: 执行重构
4. Code Review: 审查结果
```
