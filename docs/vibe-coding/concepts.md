# Vibe Coding 理念与范式

> AI 时代的编程新方式

## 核心理念

Vibe Coding（氛围编程）是一种全新的编程范式：
- **描述意图** 而不是写代码
- **AI 执行** 而不是手动编码
- **创意为主** 而不是重复劳动

## 与传统编程对比

| 方面 | 传统编程 | Vibe Coding |
|------|----------|-------------|
| 输入 | 代码规范 | 业务意图 |
| 输出 | 详细代码 | 功能实现 |
| 重点 | 实现细节 | 产品价值 |
| 角色 | 码农 | 设计师 |

## 核心理念

### 1. 描述而非编写

```
传统：写一个登录表单
Vibe：帮我做一个用户登录页面，要现代简约风格
```

### 2. 迭代而非一次

```
Vibe 流程：
1. 描述需求
2. AI 生成
3. 调整描述
4. 迭代优化
```

### 3. 验收而非实现

```
关注点：
✓ 功能是否满足需求
✓ 体验是否符合预期
✗ 具体代码怎么写
```
