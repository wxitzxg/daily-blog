# Vibe Coding 工具链

> 打造高效工作流

## 核心工具

| 工具 | 用途 |
|------|------|
| Claude Code | 核心编程助手 |
| Cursor | GUI 编程 |
| V0 | UI 生成 |
| Bolt | 全栈开发 |
| Lovart | 自动化 |

## 工具组合

### 前端开发

```
描述 → V0/Bolt → 页面
          ↓
      Cursor → 调整
          ↓
     Claude → 优化
```

### 后端开发

```
描述 → Claude → API
          ↓
     Claude → 业务逻辑
          ↓
      测试 → 验证
```

### 全栈开发

```
描述 → Bolt → 完整项目
         ↓
    Claude → 调试优化
         ↓
    部署 → 上线
```

## 实践技巧

### 1. 工具选择

| 场景 | 推荐工具 |
|------|----------|
| 页面UI | V0, Bolt |
| 复杂逻辑 | Claude Code |
| 快速原型 | Cursor |
| 自动化 | Lovart |

### 2. 工作流

```
1. Bolt 快速生成原型
2. Cursor 调整细节
3. Claude Code 深度开发
```
