# Vibe Coding 实践方法

> 从入门到精通

## 实践路径

### 阶段 1：尝试

```
任务：做一个简单页面
工具：V0 / Bolt
目标：成功生成页面
```

### 阶段 2：熟悉

```
任务：修改生成结果
工具：V0 / Cursor
目标：精准调整细节
```

### 阶段 3：掌握

```
任务：完整功能开发
工具：组合工具链
目标：高效完成项目
```

## 实践技巧

### 1. 描述清晰

```
❌ 做一个好看的页面
✓ 做一个用户登录页面，蓝白配色，圆角按钮
```

### 2. 迭代优化

```
第一版：基础功能
第二版：优化体验
第三版：完善细节
```

### 3. 验收标准

```
检查清单：
- [ ] 功能完整
- [ ] 界面美观
- [ ] 响应迅速
- [ ] 无明显 bug
```
