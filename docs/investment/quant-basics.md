# 量化投资基础

> AI 辅助投资入门

## 核心理念

量化投资是用数学模型和计算机程序进行投资决策的方法。AI 可以帮助：
- 数据分析
- 策略回测
- 风险控制

## 基本概念

| 概念 | 说明 |
|------|------|
| 量化策略 | 基于规则的交易系统 |
| 回测 | 用历史数据验证策略 |
| 回撤 | 资金从高点下跌 |
| 夏普比率 | 风险调整收益 |

## AI 在投资中的应用

### 1. 数据处理

```
AI 帮助：
- 清洗数据
- 特征工程
- 异常检测
```

### 2. 策略研发

```
AI 帮助：
- 发现规律
- 优化参数
- 组合策略
```

### 3. 风险管理

```
AI 帮助：
- 监控风险
- 预警异常
- 自动平仓
```
