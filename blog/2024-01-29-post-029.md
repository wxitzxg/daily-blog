## 🎯 claude-mem 是什么？

claude-mem 是一个 **为 Claude AI 添加长期记忆的开源框架** ，主要功能：

1. **持久化对话历史** ：所有对话永久保存
2. **智能检索** ：根据当前对话自动调取相关记忆
3. **跨会话连续性** ：下次对话时，Claude 记得你是谁
4. **个性化学习** ：根据历史对话优化回复风格

**核心价值** ：  
将 Claude 从「无状态的对话工具」升级为「有记忆的智能助手」。

## 🛠️ 技术架构

claude-mem 的核心是 **分层记忆系统** + **语义检索引擎** ：

```
┌─────────────────────────────────────┐
│         Claude API 调用层            │
└─────────────────┬───────────────────┘
                  │
┌─────────────────▼───────────────────┐
│      记忆注入层（Memory Injection） │
│  - 检索相关记忆                      │
│  - 构建增强 Prompt                   │
└─────────────────┬───────────────────┘
                  │
┌─────────────────▼───────────────────┐
│      记忆存储层（Memory Storage）    │
│  ├─ 短期记忆（当前会话）             │
│  ├─ 长期记忆（向量数据库）           │
│  └─ 元数据（标签、时间戳）           │
└─────────────────────────────────────┘
```

### 核心组件

1. **记忆编码器** （Memory Encoder）
	- 使用 **Sentence Transformers** 将对话转为向量
	- 模型： `all-MiniLM-L6-v2` （384 维向量）
2. **向量数据库** （Vector DB）
	- 支持：Pinecone、Weaviate、Chroma
	- 索引方式：HNSW（高效近似最近邻搜索）
3. **检索策略** （Retrieval Strategy）
	- 混合检索：语义相似 + 时间衰减
	- Top-K 动态调整（根据相关性分数）

## 📊 记忆检索算法

claude-mem 使用 **混合检索策略** ：

### 1\. 语义相似度（70% 权重）

```
import torch
from sentence_transformers import SentenceTransformer

model = SentenceTransformer('all-MiniLM-L6-v2')

def semantic_similarity(query, memory):
    """计算语义相似度"""
    query_embedding = model.encode(query, convert_to_tensor=True)
    memory_embedding = model.encode(memory, convert_to_tensor=True)
    
    # 余弦相似度
    similarity = torch.nn.functional.cosine_similarity(
        query_embedding.unsqueeze(0), 
        memory_embedding.unsqueeze(0)
    ).item()
    
    return similarity

# 示例
query = "如何优化数据库查询？"
memory = "realdog 偏好写原生 SQL 而不用 ORM"
print(semantic_similarity(query, memory))  # 输出：0.82
```

### 2\. 时间衰减（20% 权重）

```
import datetime
import math

def time_decay_score(memory_timestamp, decay_rate=0.95):
    """时间衰减评分"""
    now = datetime.datetime.now()
    delta_days = (now - memory_timestamp).days
    
    # 指数衰减
    score = math.exp(-delta_days / 30) * decay_rate
    return score

# 示例
old_memory_time = datetime.datetime(2026, 1, 1)
recent_memory_time = datetime.datetime(2026, 3, 15)

print(time_decay_score(old_memory_time))    # 输出：0.31（旧记忆）
print(time_decay_score(recent_memory_time)) # 输出：0.95（新记忆）
```

### 3\. 重要性评分（10% 权重）

```
def importance_boost(base_score, importance_level):
    """根据重要性提升分数"""
    return base_score * (1 + importance_level / 20)

# 示例
base = 0.75
important = importance_boost(base, importance_level=9)  # 0.75 * 1.45 = 1.09
normal = importance_boost(base, importance_level=5)     # 0.75 * 1.25 = 0.94
```

### 4\. 综合评分

```
def rank_memory(query, memory):
    """综合评分算法"""
    # 1. 语义相似度
    semantic_score = semantic_similarity(query, memory.content)
    
    # 2. 时间衰减
    time_score = time_decay_score(memory.timestamp)
    
    # 3. 重要性加成
    importance_score = importance_boost(1.0, memory.importance)
    
    # 加权综合
    final_score = (
        semantic_score * 0.7 +
        time_score * 0.2 +
        importance_score * 0.1
    )
    
    return final_score
```

### 2\. 加密存储

```
from cryptography.fernet import Fernet

# 生成密钥
key = Fernet.generate_key()
cipher = Fernet(key)

# 加密记忆内容
def save_encrypted_memory(content):
    encrypted = cipher.encrypt(content.encode())
    claude.save_memory(encrypted.decode())

# 解密记忆
def get_decrypted_memory(memory_id):
    encrypted = claude.get_memory(memory_id)
    decrypted = cipher.decrypt(encrypted.encode())
    return decrypted.decode()
```

### 3\. 敏感信息过滤

```
import re

def sanitize_memory(content):
    """移除敏感信息"""
    # 移除信用卡号
    content = re.sub(r'\d{4}[- ]?\d{4}[- ]?\d{4}[- ]?\d{4}', '[CARD]', content)
    
    # 移除邮箱
    content = re.sub(r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b', '[EMAIL]', content)
    
    # 移除电话号码
    content = re.sub(r'\b\d{3}[-.]?\d{3}[-.]?\d{4}\b', '[PHONE]', content)
    
    return content

# 保存前自动脱敏
original = "我的邮箱是 user@example.com，电话 123-456-7890"
safe = sanitize_memory(original)
claude.save_memory(safe)
```
