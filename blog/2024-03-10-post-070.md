***01***

**项目介绍  
**

  

Ruflo 是专为 Claude Code 量身打造的原生多智能体编排框架，通过蜂群式（Hive Mind）架构协调多个智能体完成软件开发任务。采用 Queen-Worker 层级调度，并支持多种共识算法自动解决多智能体协作中的冲突。

  

Ruflo 内置 **60+专业智能体** ，覆盖编码、测试、安全审计、架构设计、文档撰写等全开发链路，像真实团队一样各司其职。此外，Ruflo 通过 **MCP 协议** 无缝接入 Claude Code，用户不用离开聊天界面，就能直接召唤蜂群（多个专业智能体组成）、初始化任务、检索记忆，整个开发流程非常丝滑，还能一键解锁 170+ 专业工具。

  

****针对大模型的 “失忆” 问题，Ruflo 内置 RuVector 持久化向量记忆库，基于 PostgreSQL 与 HNSW 算法实现高速向量搜索。还结合了自进化神经架构 SONA 与 EWC++ 防遗忘技术， ****Ruflo**** 会自动提炼并存储任务经验，在后续相似需求中实现记忆复用，持续提升协作效果。****

  

性能方面，Ruflo 设计了三级智能路由降低 API 成本。简单格式调整等任务用 WASM 本地秒处理；中等任务交给轻量模型；只有真正复杂的部分才调用 Opus 等高端模型。

  

***02***

**使用**

  

环境要求：Node.js 20+（必需）npm 9+ /pnpm/bun（ 包管理器）。

  

（1）安装 Claude Code

  

```nginx
# 1. 全局安装 Claude Codenpm install -g @anthropic-ai/claude-code
# 2. （可选）跳过权限检查以加快安装配置claude --dangerously-skip-permissions
```

  

（2）npm/npx 安装：

  

```nginx
# 快速启动（无需提前安装）npx ruflo@latest init
# 或全局安装npm install -g ruflo@latestruflo init
# 使用 Bun（更快）bunx ruflo@latest init
```

  

（3）基本使用

  

```nginx
# 初始化项目npx ruflo@latest init
# 启动 MCP 服务器，用于 Claude Code 集成npx ruflo@latest mcp start
# 使用智能体运行一个任务npx ruflo@latest --agent coder --task "Implement user authentication"
# 列出所有可用智能体npx ruflo@latest --list
```

  

（4）升级

  

```cs
# 更新 helpers 和 statusline（保留你的数据）npx ruflo@v3alpha init upgrade
# 更新并自动添加缺失的技能/智能体/命令npx ruflo@v3alpha init upgrade --add-missing
```

  

（5）Claude Code MCP 集成

  

```nginx
# 将 ruflo 添加为 Claude Code 的 MCP 服务器claude mcp add ruflo -- npx -y ruflo@latest mcp start
# 验证安装claude mcp list
```

  

添加完成后，Claude Code 可直接使用 Ruflo 的全部 175+ MCP 工具，例如：

  

- swarm\_init - 初始化智能体蜂群
- agent\_spawn - 生成专业智能体
- memory\_search - 使用 HNSW 向量搜索模式
- hooks\_route - 智能任务路由
- 以及 170+ 其他工具
