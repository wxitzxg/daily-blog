## 🎯Qlib 是什么？

Qlib 是微软开源的 **AI导向量化投资平台** ，旨在利用人工智能技术赋能量化研究。它支持多种机器学习建模范式，包括：

| 📊 | **监督学习** ：挖掘市场复杂非线性模式 |
| --- | --- |
| 🔄 | **市场动态建模** ：自适应概念漂移技术 |
| 🎮 | **强化学习** ：建模连续投资决策 |
| 🤖 | **RD-Agent** ：LLM驱动的自动化R&D |

## 🚀5分钟快速开始

安装非常简单，一条命令搞定：

```
# 安装 Qlib
pip install pyqlib

# 准备数据
python -m qlib.cli.data qlib_data --target_dir ~/.qlib/qlib_data/cn_data --region cn

# 运行示例工作流
qrun benchmarks/LightGBM/workflow_config_lightgbm_Alpha158.yaml
```

运行后你将得到完整的回测分析报告，包括：

| 📈 | 年化收益率、信息比率 |
| --- | --- |
| 📊 | 最大回撤分析 |
| 🎯 | IC值、分组收益等详细指标 |
