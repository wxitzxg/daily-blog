这个开源项目，快把“AI盯盘”做成日更产品了

**正文：**

说实话，我最近看到不少“AI炒股工具”，第一反应都不太好。 不是因为它们不能用，而是太爱把自己写成“神谕机”了：好像接个大模型，就能替你看懂市场，顺手再把钱赚了。

但这个叫 **Daily Stock Analysis** 的开源项目，我看完 GitHub 之后，感觉稍微不一样一点。它没把重点全堆在“AI 多聪明”上，反而在认真处理一件更实际的事： **怎么把盯盘、复盘、推送这套很碎的流程，压成一套每天能自动跑的东西。** 项目本身支持 A 股、港股、美股，分析完会把结果推到企业微信、飞书、Telegram 或邮箱里，默认还能通过 GitHub Actions 定时执行，官方甚至把门槛写得很直白：5 分钟部署，不用服务器。

它最抓人的点，不是“支持多少模型”，而是那个输出形式。 不是长篇大论，也不是把技术指标铺一脸，而是先压成一句核心判断，再给买入价、止损价、目标价，后面再补操作检查清单。这个味道很像把散户每天最耗神的那一段——看 K 线、翻新闻、猜情绪、定纪律——先替你做了个预处理。README 里写得也很直接：技术面、筹码分布、舆情情报和实时行情会被放到一起分析，另外还有“严禁追高”、乖离率阈值、趋势排列这些纪律约束，不是单纯给一个“看多看空”就算完。

我觉得这项目真正有意思的地方，其实是它已经开始往“半自动研究台”那个方向长了。 一边是市场策略系统，A 股内置三段式复盘策略，美股有 Regime Strategy，最后会输出进攻、均衡、防守，或者 risk-on、neutral、risk-off 这样的计划；另一边又接了 Agent 问股，能用均线金叉、缠论、波浪理论这些策略多轮对话。你甚至还能直接扔一张自选股截图进去，让 Vision 模型把代码识别出来加到监控列表里。这个组合放在一起，已经不只是“帮你看一只票”了，更像是想把个人投资者每天那套重复劳动打包掉。

还有一个我比较在意的小细节： 它不是只写了个 README 就停了，最近几周还在持续加东西。2 月底项目接了 LiteLLM，把 Gemini、Claude、OpenAI 兼容接口、DeepSeek 这些模型统一起来，还支持多 Key 轮转；3 月初又在修桌面端和 WebUI 的一堆问题。你能明显感觉到，这不是“发出来蹭个 AI 概念”的仓库，而是作者真的在把它往长期可用的产品感上拧。

当然，这类工具最危险的地方也恰恰在这里。 它会让人产生一种很舒服的幻觉：决策已经被整理好了，我只需要照着做就行。可项目自己也写了“仅供参考，不构成投资建议”，这句不能当免责声明看完就划走，它其实是在提醒一件很朴素的事： **AI 可以帮你省时间，但它没法替你承担风险。** 尤其是这种会直接给买入价、止损价、目标价的系统，越像仪表盘，越容易让人误以为它已经替你完成了判断。

所以如果非要我给它下个判断，我会说： 这不是一个“AI 帮你赚钱”的故事，它更像一个把股票研究流程产品化的开源项目。对于本来就有自己交易框架的人，它可能挺省事；对于想靠一句 AI 结论直接上车的人，反而要小心一点。

毕竟市场最爱收拾的，往往就是那种看起来已经被解释清楚的时候。
