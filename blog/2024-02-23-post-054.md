## 项目简介

`learn-claude-code` 通过对 Claude Code 的深度拆解，教你如何从一个最简单的循环开始，一步步构建出一个优雅、强大的智能体载具系统，让 AI 在你的环境里真正拥有手和眼。

开篇先提出了一个观点： **模型就是 Agent，而你的代码只是 Harness（载具）** 。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

简单来说，智能是学出来的，不是编出来的。像 Claude 这样的模型本身就是那个身经百战的驾驶员，而开发者真正该做的工作，是为它造一辆性能卓越的赛车，也就是给它提供环境、工具、感官和权限。

## 核心内容

这个项目最有价值的地方，在于它把高深的 Agent 开发拆解成了 12 个递进式的工程版本。

在前期阶段，它会帮你打破对 AI 的神秘感。比如 S01 阶段，它只用一个基础循环加上 Bash 工具，就让模型具备了在你的电脑上行动的能力。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

随着课程的深入，它提供了一系列能让你一个人顶一个团队的大招。

它会教你如何通过 S03 的 Todo 管理让 AI 谋定而后动，解决走哪算哪的盲目感。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

## 学习方式

想要把这些资源转化成战斗力，你可以通过极其简单的指令快速配置好你的实验环境。

首先克隆仓库并安装必要的依赖，记得在环境变量中填入你的 ANTHROPIC API 密钥：

```
git clone https://github.com/shareAI-lab/learn-claude-code
cd learn-claude-code
pip install -r requirements.txt
cp .env.example .env   # 编辑 .env 填入你的 API 密钥
```

真正的学习不是直接运行最终版本，而是观察代码的演进过程 **。建议你按照 S01 到 S12 的顺序逐个运行脚本，观察载具机制是如何一层层叠加的。**

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

你可以从最基础的循环开始尝试，一直演进到具备完整隔离能力的终点：

```
python agents/s01_agent_loop.py       # 最小循环起点
python agents/s12_worktree_task_isolation.py  # 完整递进终点
```

### 交互式 Web 平台

如果你觉得直接看 Python 脚本不够直观，项目还贴心地准备了一个基于 Next.js 的 Web 交互平台。

在这里，你可以看到每个课程对应的分步动画和源码查看器，这种可视化的方式能帮你更清晰地建立起对 Agent 载具运作逻辑的肌肉记忆。

![图片](data:image/svg+xml,%3C%3Fxml version='1.0' encoding='UTF-8'%3F%3E%3Csvg width='1px' height='1px' viewBox='0 0 1 1' version='1.1' xmlns='http://www.w3.org/2000/svg' xmlns:xlink='http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke='none' stroke-width='1' fill='none' fill-rule='evenodd' fill-opacity='0'%3E%3Cg transform='translate(-249.000000, -126.000000)' fill='%23FFFFFF'%3E%3Crect x='249' y='126' width='1' height='1'%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)
