## 项目简介

`Lightpanda` 是一款完全从零开始构建的开源无头浏览器，专为AI代理、网页抓取和自动化测试而生。它没有基于Chromium或WebKit，而是用Zig语言重写了核心引擎，主打“极致性能”。

## 项目体验展示

开源君在 Google Colab 简单用 playwright 跑了一下 Lightpanda，整体体验可以用一句话概括：又快又轻，非常适合高并发数据抓取和 AI Agent 网页解析场景。

### 测试一：极致加载速度测试（验证“秒开”特性）

测试一下，从拉起底层进程、连接浏览器到抓取完 GitHub 的网页标题，一共需要多久。

```
const { chromium } = require('playwright-core');
const { lightpanda } = require('@lightpanda/browser');

(async () => {
const start = Date.now();

// 瞬间启动底层的 Lightpanda 进程
const proc = await lightpanda.serve({ host: '127.0.0.1', port: 9222 });
const browser = await chromium.connectOverCDP('http://127.0.0.1:9222');
const context = await browser.newContext();
const page = await context.newPage();

await page.goto('https://github.com/lightpanda-io/browser');
const title = await page.title();

console.log(\`抓取成功: ${title}\`);
console.log(\`总耗时: ${Date.now() - start}ms\`);

await browser.close();
  proc.kill();
})();
```

### 测试二：模拟交互操作（避开视觉渲染限制）

既然它去掉了复杂的视觉渲染，那还能不能交互呢？答案是完全可以。

```
const { chromium } = require('playwright-core');
const { lightpanda } = require('@lightpanda/browser');

(async () => {
const proc = await lightpanda.serve({ host: '127.0.0.1', port: 9223 });
const browser = await chromium.connectOverCDP('http://127.0.0.1:9223');
const context = await browser.newContext();
const page = await context.newPage();

await page.goto('https://www.wikipedia.org/');

// 在搜索框填入内容
await page.locator('#searchInput').fill('Zig language');

// 点击事件
await page.locator('button[type="submit"]').evaluate(btn => btn.click());
console.log('已触发搜索点击');

await page.waitForTimeout(3000); 
console.log(\`跳转后的 URL: ${page.url()}\`);

await browser.close();
  proc.kill();
})();
```

### 测试三：高并发压力测试（验证低资源占用）

直接用 `Promise.all` 瞬间并发拉起 10 个独立的无头浏览器实例。

```
const { chromium } = require('playwright-core');
const { lightpanda } = require('@lightpanda/browser');

asyncfunction fastFetch(id) {
const port = 9224 + id; 
const proc = await lightpanda.serve({ host: '127.0.0.1', port });
const browser = await chromium.connectOverCDP(\`http://127.0.0.1:${port}\`);
const context = await browser.newContext();
const page = await context.newPage();

await page.goto('https://example.com');
const title = await page.title();

await browser.close();
  proc.kill();
return\`任务 ${id} 完成: ${title}\`;
}

(async () => {
console.time('10并发总耗时');
// 同时启动 10 个实例去抓取
const tasks = Array.from({ length: 10 }, (_, i) => fastFetch(i));
const results = awaitPromise.all(tasks);

console.log(results.join('\n'));
console.timeEnd('10并发总耗时');
})();
```

## 功能特性

- 无传统内核依赖，基于 Zig 语言开发，摒弃图形渲染层，专为无头场景优化
- 兼容主流自动化工具，通过 CDP 协议无缝对接 Playwright、Puppeteer、chromedp
- 完善的 Web 基础能力，支持 JS 执行、DOM 树解析、Ajax/XHR/Fetch 等核心 API
- 丰富的自动化功能，涵盖点击、表单输入、Cookies 管理、代理支持和网络拦截
- 实用的个性化配置，支持自定义 HTTP 头、遵守 robots.txt 规则，可禁用遥测数据采集

## 快速安装、使用

Lightpanda支持Linux x86\_64、MacOS aarch64系统，Windows用户可通过WSL2运行，同时提供官方Docker镜像，两种安装方式任选：

### 二进制安装

Linux系统：

```
curl -L -o lightpanda https://github.com/lightpanda-io/browser/releases/download/nightly/lightpanda-x86_64-linux && chmod a+x ./lightpanda
```

MacOS系统：

```
curl -L -o lightpanda https://github.com/lightpanda-io/browser/releases/download/nightly/lightpanda-aarch64-macos && chmod a+x ./lightpanda
```

### Docker安装（推荐生产环境）

```
docker run -d --name lightpanda -p 9222:9222 lightpanda/browser:nightly
```

### 基础使用

1. 直接抓取URL内容
```
./lightpanda fetch --obey_robots --log_format pretty --log_level info 目标URL
```
1. 启动CDP服务器
```
./lightpanda serve --obey_robots --log_format pretty --log_level info --host 127.0.0.1 --port 9222
```
1. 对接Puppeteer（脚本无需修改，仅需指定连接地址）：
```
'use strict'
import puppeteer from 'puppeteer-core';
const browser = await puppeteer.connect({
  browserWSEndpoint: "ws://127.0.0.1:9222",
});
const page = await browser.newPage();
await page.goto('https://example.com',{ waitUntil: "networkidle0" });
console.log(await page.title());
await page.close();
await browser.disconnect();
```
