#### 这到底是个什么神仙项目？

这个项目的名字就叫 **skills** ，来自 MiniMax-AI 团队。

简单来说，它是一套开发技能包，专门为 AI 编码助手设计的。你可以把它直接装到 Claude Code、Cursor、Windsurf、OpenCode 这些你常用的 AI 编程工具里，然后 AI 的代码质量直接起飞！

它是一套 **结构化的、生产级的开发工作流** ——从需求分析、架构设计，到具体实现，AI 照着这套流程走，输出的代码质量直接对标大厂资深工程师。

涵盖了前端、全栈、Android、iOS、办公等多个领域，每个领域都有专门的技能包，把资深工程师多年积累的经验都打包进去了。

#### 核心技能包

1️⃣ **frontend-dev** - 前端开发的终极武器

这个技能包简直是前端开发者的福音！它不是简单地教你写 React 组件，而是一套完整的全栈前端开发方案。

技术栈涵盖 React / Next.js + Tailwind CSS，这是现在最流行的组合。但厉害的地方在于，它还集成了：

- • **电影级动画** ：Framer Motion、GSAP，让你的网页动效丝滑得像电影一样
- • **AI 生成媒体素材** ：通过 MiniMax API 直接生成图像、视频、音频、音乐、TTS，再也不用到处找素材了
- • **说服性文案** ：AIDA 框架，连文案都帮你写好了
- • **生成艺术** ：p5.js、Three.js、Canvas，想整点炫酷的视觉效果？小菜一碟

2️⃣ **fullstack-dev** - 全栈开发的最佳实践

如果你要做全栈项目，这个技能包绝对是你的得力助手。它涵盖了：

- • **后端架构** ：REST API 设计、认证流程（JWT、session、OAuth）
- • **实时功能** ：SSE、WebSocket，做实时通讯、实时更新都不在话下
- • **数据库集成** ：SQL / NoSQL 都支持
- • **生产加固** ：上线前的检查清单，帮你避免各种坑
- • **指导工作流** ：从需求 → 架构 → 实现，一步步带你走

3️⃣ **android-native-dev** - Android 原生开发专家

做 Android 原生开发的朋友看过来！这个技能包基于 Material Design 3，涵盖：

- • **Kotlin / Jetpack Compose** ：现代 Android 开发的标配
- • **自适应布局** ：适配各种屏幕尺寸
- • **Gradle 配置** ：再也不用为 Gradle 头疼了
- • **无障碍支持** ：WCAG 标准，让你的应用对所有人都友好
- • **构建排错** ：遇到编译问题？它来帮你
- • **性能优化** ：让你的应用跑得飞快
- • **动效系统** ：Material Design 的动效，让应用体验更上一层楼

4️⃣ **ios-application-dev** - iOS 应用开发指南

iOS 开发者也不例外！这个技能包涵盖了：

- • **UIKit、SnapKit、SwiftUI** ：不管你用哪种方式写 iOS，它都能帮你
- • **触摸目标、安全区域** ：iOS 开发的基本素养
- • **导航模式** ：iOS 风格的导航设计
- • **Dynamic Type、Dark Mode** ：适配各种系统设置
- • **无障碍支持** ：让你的应用 accessible
- • **集合视图** ：UICollectionView 的最佳实践
- • **Apple HIG 合规** ：确保你的应用符合苹果的设计规范

5️⃣ **shader-dev** - GLSL Shader 狂魔专用

这个是给图形开发者准备的！它提供了全面的 GLSL shader 技术，包括：

- • **光线步进** ：ray marching，做 3D 效果的神器
- • **SDF 建模** ：用数学函数建模
- • **流体模拟** ：炫酷的流体效果
- • **粒子系统** ：各种粒子特效
- • **程序化生成** ：用代码生成纹理、模型
- • **光照** ：各种光照算法
- • **后处理** ：让你的画面更有质感
- • **ShaderToy 兼容** ：可以直接用 ShaderToy 上的代码

6️⃣ **办公神器套件** - PDF、PPTX、Excel 一键生成

除了开发技能，这个项目还包含了几个办公神器：

**gif-sticker-maker** ：把照片（人、宠物、物体、Logo）转换成 4 张带字幕的 GIF 动图贴纸，Funko Pop / Pop Mart 风格，用 MiniMax 图像和视频生成 API 驱动。

**minimax-pdf** ：生成、填充、重新格式化 PDF 文档，有 token 化的设计系统。可以从头创建（15 种封面风格）、填充现有表单字段、或者把文档重新格式化成新设计。打印就绪的输出，排版和颜色根据文档类型自动调整。

**pptx-generator** ：生成、编辑、读取 PowerPoint 演示文稿。可以用 PptxGenJS 从头创建（封面、目录、内容、分节、总结幻灯片），通过 XML 工作流编辑现有 PPTX，或者用 markitdown 提取文本。

**minimax-xlsx** ：打开、创建、读取、分析、编辑、验证 Excel/电子表格文件（.xlsx、.xlsm、.csv、.tsv）。涵盖通过 XML 模板从头创建新的 xlsx、用 pandas 读取和分析、零格式丢失地编辑现有文件、公式重新计算、验证、专业财务格式化。

**minimax-docx** ：使用 OpenXML SDK（.NET）创建、编辑、格式化专业 DOCX 文档。三个工作流：从头创建新文档、填充/编辑现有文档的内容、或者通过 XSD 验证关卡应用模板格式化。

#### 快速安装

这个技能包支持 Claude Code、Cursor、Codex、OpenCode 等多个工具，安装方法都很简单：

Claude Code 安装：

```
claude plugin marketplace add https://github.com/MiniMax-AI/skills
claude plugin install minimax-skills
```

Cursor 安装：

```
git clone https://github.com/MiniMax-AI/skills.git ~/.cursor/minimax-skills
```

然后在 Cursor 设置里，把 skills 路径指向 `~/.cursor/minimax-skills/skills/` 就可以了。

Codex 安装：

```
git clone https://github.com/MiniMax-AI/skills.git ~/.codex/minimax-skills
mkdir -p ~/.agents/skills
ln -s ~/.codex/minimax-skills/skills ~/.agents/skills/minimax-skills
```

重启 Codex 就能发现这些 skills 了。Windows 用户可以看 `.codex/INSTALL.md` 获取详细说明。

OpenCode 安装：

```
git clone https://github.com/MiniMax-AI/skills.git ~/.minimax-skills
mkdir -p ~/.config/opencode/skills
ln -s ~/.minimax-skills/skills/* ~/.config/opencode/skills/
```

重启 OpenCode 就能用了。详细说明看 `.opencode/INSTALL.md` 。
